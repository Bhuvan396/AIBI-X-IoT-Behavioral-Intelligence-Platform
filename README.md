# AIBI-X  
### Adaptive IoT Behavioral Intelligence & Trust Analytics Platform

AIBI-X is an IoT cybersecurity analytics platform designed to monitor device behavior using network telemetry and machine learning.  
The system analyzes communication patterns of IoT devices, detects behavioral anomalies, and generates a dynamic trust score representing the security posture of each device in the network.

Instead of relying on traditional signature-based detection, the platform focuses on behavioral analysis. By studying how devices communicate across the network, AIBI-X can identify suspicious activity, detect policy violations, and highlight potential cyber threats affecting IoT environments.

---

## Key Features

- Behavioral analysis of IoT device communication using network telemetry  
- Dynamic trust score evaluation for continuous device risk assessment  
- Machine learning based detection of abnormal network behavior  
- Detection of security events such as reconnaissance, exfiltration, C2 beaconing, policy violations, and behavioral drift  
- Explainable AI insights highlighting the behavioral indicators responsible for each detection  
- Interactive dashboard for monitoring device behavior and security analytics  

---

## System Workflow

The platform operates through a telemetry-driven pipeline:

```
Device Telemetry / Attack Simulation
        ↓
Telemetry Ingestion API
        ↓
telemetry.csv
        ↓
Feature Extraction
        ↓
Machine Learning Detection
        ↓
Trust Score Computation
        ↓
Explainable Security Analysis
        ↓
Security Dashboard
```

Telemetry generated by IoT devices is stored and processed to extract behavioral features.  
Machine learning models evaluate these features to identify anomalies and classify potential threats.  
The system then computes a trust score and provides explainable indicators describing the observed behavior.

## Running the Project

### Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/AIBI-X.git
cd AIBI-X
```

### Start the backend server

```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload
```

Backend will run on:

```
http://localhost:8000
```

### Start the frontend

```bash
cd frontend
npm install
npm run dev
```

Frontend will run on:

```
http://localhost:5173
```

## Detection Workflow Example

1. Telemetry data from IoT devices is generated or injected.  
2. Telemetry records are appended to `telemetry.csv`.  
3. Behavioral features are extracted from telemetry data.  
4. Machine learning models evaluate the features to detect anomalies.  
5. The system classifies potential attack patterns and computes a trust score.  
6. Explainable AI provides behavioral indicators and analysis through the dashboard.
